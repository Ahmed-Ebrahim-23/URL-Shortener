## Quick Start

To get started quickly, run `docker compose up` in the project root directory. This will start the backend API, frontend UI, and Redis cache.

## Prerequisites

- Docker and Docker Compose

## Setup Instructions

1. Clone the repository `git clone https://github.com/Ahmed-Ebrahim-23/URL-Shortener.git`.
2. Navigate to the project directory.
3. Run `docker compose up` to build and start all services.
4. Access the frontend at `http://localhost:5173` and backend API at `http://localhost:3000`.

## Technology Stack

- **Backend:** Node.js, Express.js, MongoDB, Redis
- **Frontend:** React, Vite
- **Containerization:** Docker, Docker Compose
